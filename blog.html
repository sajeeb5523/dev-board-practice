Question-1: What are the different ways to select an element in the DOM?

Elements can be selected using getElementById, getElementsByClassName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML manages HTML tags, innerText shows only visible text, and textContent shows all text including hidden ones.

Question-3: What is event delegation in the DOM?

It is a technique of adding a single event listener to a parent element to manage events for all its current and future child elements.

Question-4: What is event bubbling in the DOM?

It is a process where an event starts from the target element and moves up to its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Use createElement() to create, appendChild() to add, and remove() or removeChild() to delete an element.
